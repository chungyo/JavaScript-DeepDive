# 🧠 JavaScript-DeepDive

이 레포지토리는 프론트엔드 개발자로서 코딩 테스트를 준비하기 위한 알고리즘 풀이 기록 공간입니다.  
모든 문제는 JavaScript로 풀이하며, 문제 유형별로 정리하여 실력을 점진적으로 향상시키는 것을 목표로 합니다.

## 📌 목표
- 매일 1~2문제 이상 꾸준히 풀이
- 문제 풀이 후 시간복잡도 & 개선 포인트 정리
- JS 문법과 알고리즘 패턴 모두 챙기기
